# 2026.07.13
유니티 프로그래밍 수업 포트폴리오 과제

## 진행 방법
### Assets > TitleScene 실행 > Play > Start 버튼 클릭 > 직업 선택 > 게임 시작 > 플레이어 사망 시 게임 종료

플레이어는 곳곳에서 생성되어 플레이어에게 다가오는 몬스터들을 처치하여 레벨업하면서 최대한 오래 살아남는 게임.

(현재 전사만 구현. 궁수, 마법사도 선택은 가능. 직업도 궁수, 마법사로 변경되지만 현재 플레이어 스킬은 전사 스킬로 고정됨.)

##  기능

### 1. 캐릭터 이동(WASD), 캐릭터 컨트롤러를 통해 설정

### 2. 화살표 (←→)를 이용한 회전

### 3. 메인 카메라를 시네머신에 연결하여 플레이어의 뒤에 위치하고 이동, 회전할 경우 카메라도 따라서 이동, 회전하도록 설정

### 4. Q, E를 통해 현재 설정된 목표 변경 가능

### 5. 1, 2, 3, 4, 5번으로 스킬 사용 가능 (전사 스킬만 구현. 다른 직업을 석택해도 전사 스킬이 선택됨. 추후 수정.)

### 6번은 궁극기. 다른 스킬들 사용 시 궁극기 게이지 상승. 게이지가 100%가 되었을 때 사용 가능. 사용 시 게이지 0으로 초기화

### 6. 플레이어 캐릭터의 외형, 애니메이션 적용 (대기, 이동, 공격, 스킬(부분 적용))

## 설명

### 플레이어를 기준으로 몬스터들이 설정한 범위 내에서 랜덤한 위치에 생성.
### 플레이어를 목표로 이동, 근접 시 공격.
### 플레이어는 WASD 외에도 스페이스바를 이용해 현재 목표로 설정된 몬스터를 향해 자동 이동 가능
### QE로 목표를 변경 가능.
### 1, 2, 3, 4, 5 입력 시 스킬 발동(사거리 내에 목표가 존재할 경우. 목표가 사거리 밖에 있는 경우에는 목표에게 접근 후 스킬 시전)
